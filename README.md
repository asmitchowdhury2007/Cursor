# Cursor
A front-end clone of the Cursor website, built with HTML and CSS. This project replicates the layout, design system,  to improve front-end development skills.
This project is a frontend recreation of the Cursor website interface. The goal of this clone was to practice modern web development concepts such as  Flexbox, CSS Grid, positioning, hover effects, spacing, typography, and clean UI design.

Features:
1.Navigation bar
2.Modern UI layout inspired by Cursor
3.CSS Flexbox and Grid implementation
4. cards and sections
5.Hover effects and transitions
 
Technologies Used:

1.HTML5
2.CSS3
